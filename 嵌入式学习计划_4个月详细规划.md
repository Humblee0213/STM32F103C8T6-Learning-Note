 ## 第五阶段：ARM Cortex-M3内核架构（第12周）

 **目标：** 理解CM3内核，为RTOS底层原理分析做准备。

 ### 第12周（8.29 周六 - 9.4 周五）

 | 日期 | 学什么 | 做什么 |
 |------|--------|--------|
 | **8.29（周六）** | CM3第1-2章+第3章（上） | 架构演进概览/寄存器组/通用寄存器R0-R15 |
 | **8.30（周日）** | CM3第3章（下）：基础 | 特殊功能寄存器/操作模式/异常类型/向量表/堆栈/复位序列 |
 | **8.31（周一）** | CM3第4章：汇编语言基础 | 常用ARM汇编指令：LDR/STR/MOV/B/BL/PUSH/POP/CPS |
 | **9.1（周二）** | CM3第5章：存储器系统 | 存储器映射/位带操作/非对齐访问/总线接口 |
 | **9.2（周三）** | CM3第6章 + 第7章 | 流水线/SVC和PendSV（RTOS任务切换关键） |
 | **9.3（周四）** | CM3第8章：NVIC + 第9章 | 中断优先级管理/悬起/异常响应序列/入栈出栈/咬尾中断/晚到中断 |
 | **9.4（周五）** | CM3第13章：SysTick定时器 | SysTick配置/作为RTOS时钟/电源管理（RTOS时钟基准） |

 **核心笔记：**
 - SVC（系统服务调用）：应用程序通过SVC触发进入特权模式，RTOS调度入口
 - PendSV（可挂起系统调用）：RTOS任务切换的核心机制——把任务切换延迟到所有ISR处理完成之后
 - SysTick定时器：RTOS的心跳，周期性产生中断驱动调度器
 - 中断入栈机制：硬件自动压栈8个寄存器（xPSR/PC/LR/R12/R3-R0），理解这个对RTOS任务堆栈初始化非常重要

 **里程碑检查：** 理解了CM3内核架构、异常机制、任务切换涉及的硬件行为
